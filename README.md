# 🔐 Passport.js Authentication System

A secure **Authentication System** built using **Node.js, Express.js, MongoDB, Passport.js, Express Session, EJS, Bootstrap, and bcrypt**.

This project demonstrates user authentication using **Passport.js Local Strategy**, allowing users to register, log in securely, manage sessions, and access protected routes.
